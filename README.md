# SkillForge 🛠️
### Forge Your Skills. Own Your Story.

SkillForge is a student skill showcase platform where students can display their skills, projects, achievements and get discovered by recruiters.

## 🚀 Features
- Student Register & Login (JWT Auth)
- Profile setup with completion score
- Add Skills with proficiency levels & skill bars
- Showcase Projects with live links
- Add Achievements & Internships
- Peer Skill Endorsement (LinkedIn style)
- Auto-generated Resume PDF download
- Public shareable profile page
- Search & discover students by skill
- Mobile Responsive Design

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Auth:** JWT + Bcrypt
- **PDF:** Puppeteer

## ⚙️ Setup Instructions

1. Clone the repository
   git clone https://github.com/yourusername/SkillForge.git

2. Install dependencies
   npm install

3. Create .env file
   DB_PASSWORD=your_mysql_password
   JWT_SECRET=skillforge_secret_key
   PORT=3000

4. Setup MySQL database and run SQL queries

5. Start the server
   node server.js

6. Open browser
   http://localhost:3000
